Approximate Duplicate Detection

Description

Options

The following table describes the options associated with the Approximate Duplicate Detection step

Option Description
Step Name The name of this step as it appears in the transformation workspace. This name must be unique within the transformation.
Grouping Column Name The name of the output column that contains the group to each the record belongs to.
Similarity Column Name The name of the output column that contains the similarity between the first record of the group and the record in the row.
Remove Non-Duplicates By selecting this option, the records that do not match any other records are removed from the step output.
